Tentang Lubuntu

Lubuntu adalah sistim operasi yang ringan dan dengan user interface yang mudah digunakan. Berbasis Linux dengan menggunakan desktop LXDE/LXQT, dan pilihan aplikasi aplikasi yang ringan, karena ini Lubuntu membutuhkan spesifikasi hardware yang rendah. Lubuntu diawali oleh Mario Behling dan telah bertumbuh selama beberapa tahun oleh Julien Lavergne[1]

Sejarah

Lubuntu pertama release di sekitar tahun 2008 bersamaan saat keluarnya ubuntu 8.10 Intrepid Ibex, namun tidak dalam bentuk terpisah dan hanya bisa di install sebagai package dari repositori ubuntu [5], Lubuntu menerima status sebagai derivative ubuntu resmi bersamaan dengan keluarnya ubuntu 11.10 di sekitar tahun 2011 [6]
